HOW TO USE:

  1. Open your web-browser's console (on Chrome you press F12)
  2. Explore! use
    Docs(thing to get docs on);
    to get some (super basic) documentation on a variable or method.
  3. Look at the examples below for some more info.



EXAMPLES:

Example 1 - Firebolt

var myFocus = new Focus().init(); //The shorthand way to initialize your Focus. There are other ways, but they are longer.

var longFire = new Node(types.effect,effects.fire).addElement("setting.duration",5).addElement("setting.power",5); //This makes the effected mob get lit on fire for 5 seconds (the maximum)

myFocus.addNode(new Node(types.medium,mediums.bolt));//Thunderbolt-styled focus
myFocus.addNode(longFire);//...........................With a firey effect!

myFocus.command(); //Export to a /summon command

Example 2 - Fluxbats

var myFocus = new Focus().init();

myFocus.addNode(new Node(types.medium,mediums.touch)); //Just so the bat doesn't spawn right on your face

var bats = new Node(types.medium,medium.bat).addElement("setting.target",0);
var flux = new Node(types.effect,effects.flux).addElement("setting.power",5);

myFocus.addNode(bats);
myFocus.addNode(flux);

myFocus.command();
